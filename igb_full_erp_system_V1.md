IGB 全功能 ERP 系統 V1.0.0 初始版本發布備忘錄文件名稱: igb_full_erp_system_V1.html版本號: V1.0.0 (初始發布)發布日期: 2025年10月26日文件摘要: 本文件標誌著 IGB 企業資源規劃（ERP）系統的第一個完整版本發布。此 HTML 文件包含所有核心模組的前端介面，用於測試、演示及用戶培訓。一、版本目的與範圍本 V1.0.0 版本的核心目的在於建立一套集成的、可運行的 ERP 系統基礎框架，涵蓋企業運營中最關鍵的流程。主要功能範圍包括：用戶認證與權限管理: 提供登入/登出功能，並能基於角色進行功能介面權限劃分。主數據管理 (Master Data): 員工、客戶、供應商、物料清單 (BOM) 等基礎數據的錄入與維護。庫存管理 (Inventory): 包含入庫、出庫、庫存查詢和盤點功能。採購流程 (Procurement): 涵蓋請購單、採購訂單的建立、審批及接收流程。銷售管理 (Sales): 包含報價、銷售訂單建立及出貨流程。二、技術與環境文件格式: 單一 HTML 檔案，包含所有 HTML 結構、CSS 樣式（使用 Tailwind CSS）及 JavaScript 邏輯。數據模型: 系統邏輯已準備好與後端 API 服務進行數據交換（例如：Firebase/Firestore 或模擬 API）。三、注意事項本版本為系統的首次穩定發布，旨在提供一個全面的功能展示和使用平台。所有後續的功能增強、性能優化及介面調整將在 V1.1 及更高版本中進行迭代更新。發布人: [系統開發團隊]
