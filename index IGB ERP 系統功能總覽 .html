Enterprise Resource Planning System

IGB ERP 系統功能總覽

IGB ERP 系統旨在整合企業內部所有資源和流程，提供從財務、供應鏈到人事的全面解決方案。透過模組化的設計，企業可以根據需求彈性配置，並利用雲端架構實現高效營運。

七大核心模組，全面掌控營運

核心營運與供應鏈管理 (進銷存)

進銷存 (Sales, Purchasing, Inventory Management)

功能描述：整合採購、銷售與庫存作業流程。從請購、採購單、入庫，到銷貨訂單、出貨、應收/應付帳款的產生，實現物流、資訊流與資金流的同步。

🚀 主要效益

  • ✔ 提升庫存準確性
  • ✔ 優化採購成本
  • ✔ 加速訂單處理週期

財務會計管理

財務會計 (Financial Accounting)

功能描述：處理企業所有會計事務，包括總帳、應收帳款、應付帳款、固定資產及成本會計。支援自動拋轉傳票，生成標準會計報表（如資產負債表、損益表）。

🚀 主要效益

  • ✔ 確保財務數據的準確與即時性
  • ✔ 符合法規要求
  • ✔ 提供可靠的財務分析

基礎資料與主檔管理

商品管理 (Product Management)

功能描述：集中管理所有銷售和採購商品的基礎資料。包含商品編號、規格、單位、成本、定價策略、安全庫存量等資訊，是進銷存作業的基礎。

🚀 主要效益

  • ✔ 統一商品標準
  • ✔ 避免資料重複與錯誤
  • ✔ 提升營運效率

廠商管理 (Vendor Management)

功能描述：管理所有供應商和承包商的詳細資訊。包括聯繫方式、交易歷史、採購價格、付款條件、信用額度評估等。

🚀 主要效益

  • ✔ 建立穩定的供應鏈關係
  • ✔ 有效評估與篩選供應商
  • ✔ 控管採購風險

人力資源管理

人事管理 (HR Management)

功能描述：涵蓋員工資料、薪資計算、考勤管理、休假管理、績效評估等功能。協助企業有效管理人力資源，確保薪資發放與法規遵循。

🚀 主要效益

  • ✔ 提高人資部門的行政效率
  • ✔ 優化員工體驗
  • ✔ 支援人才發展

系統與雲端架構

ERP 管理 (ERP Administration)

功能描述：系統配置與權限管理中心。負責設定各模組的參數、維護主資料、定義使用者角色及操作權限，確保系統安全與穩定運行。

🚀 主要效益

  • ✔ 實現精細化的權限控管
  • ✔ 系統配置貼合企業實際需求
  • ✔ 確保系統安全與穩定運行

ERP 雲端 (Cloud ERP)

功能描述：IGB ERP 系統基於雲端架構運行，無需投入大量硬體成本，透過瀏覽器即可隨時隨地存取系統。提供自動更新、資料備份與高可用性保障。

🚀 主要效益

  • ✔ 降低IT維護成本
  • ✔ 加速部署
  • ✔ 提供業務連續性和行動工作能力
